Expected Queries During Sessions

Common questions during Welcome Webinar & Live Sessions — with recommended handling approach.

Welcome Webinar Queries
When do live sessions start? → Confirm batch start date & schedule
How do I access the platform? → Direct to upgrad.com/lxp
Who is the faculty? → IIT KGP professors — brief introduction
Is attendance mandatory? → 75% minimum required (Not applicable for 31st March 2026 batch)
Will sessions be recorded? → Yes, available post-class on Prism
How do I submit assignments? → Via Prism LMS module
Live Session Queries
Zoom link not working → Check Prism live learning tab/ raise to program manager
Audio / video issues → Rejoin · check internet / device settings
Missed session — how to catch up? → Recording available within 4-6 hrs
How to ask a question during session? → Use Zoom chat · Q&A last 15 mins
Assignment deadline extension? → No extensions — firm policy
Certificate / transcript queries → Post-completion · support team
💡 Recommended approach: Address live queries promptly via chat or in Q&A window.
